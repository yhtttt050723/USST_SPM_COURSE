# 项目管理与过程改进课程 - 需求简表

## 1. 项目概述
| 维度 | 说明 |
| --- | --- |
| 项目 | 项目管理与过程改进课程教学平台 |
| 服务对象 | 教师、学生 |
| 范围 | 课程信息、作业与出勤、讨论互动、统计展示 |
| 技术 | Vue 3 + Vite + Element Plus / Spring Boot 3.5.6 + MyBatis Plus / MySQL 8.0 |

## 2. 迭代安排
| 迭代 | 时间 | 目标 | 关键交付 |
| --- | --- | --- | --- |
| 迭代 1 | Day 1-14 | 学生端（登录→课程→作业→签到）
| 迭代 2 | Day 15-28 | 教师后台 + 讨论区 + 公告

## 3. 功能模块
1. **认证/用户**：登录、角色（学生/教师/管理员）、个人资料、账号状态管理。  
2. **课程/班级**：课程信息、班级成员、公告（迭代 2）。  
3. **作业/成绩**：发布、提交（文本/文件）、批改、成绩查询与统计。  
4. **出勤**：考勤任务、签到/签退、请假、出勤统计导出。  
5. **讨论/通知（迭代 2）**：讨论帖、评论、公告推送、系统通知。  
6. **统计/演示**：学生学习概览、教师班级报表、周报与 PPT。
