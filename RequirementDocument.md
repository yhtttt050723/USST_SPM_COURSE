# 项目管理与过程改进课程 - 需求简表

## 1. 项目概述
| 维度 | 说明 |
| --- | --- |
| 项目 | 项目管理与过程改进课程教学平台 |
| 服务对象 | 1位教师、1个班级的学生 |
| 范围 | 课程信息、作业与出勤、讨论互动、统计展示 |
| 技术 | Vue 3 + Vite + Element Plus / Spring Boot 3.5.6 + MyBatis Plus / MySQL 8.0 |
| 登录方式 | 本地账号体系（学号+密码，可注册） |

## 2. 迭代安排
| 迭代 | 时间 | 目标 | 关键交付 |
| --- | --- | --- | --- |
| 迭代 1 | Day 1-14 | 学生端（登录→课程→作业→签到） | 前/后端可运行、登录注册、核心SQL、Swagger/ApiFox、≥10条测试用例 |
| 迭代 2 | Day 15-28 | 教师后台 + 讨论区 + 公告 | 教师批改、考勤管理、讨论区与公告、统计导出、PPT/演示视频 |

## 3. 功能模块

### 3.1 认证/用户
- **登录注册**：前端提交学号+密码；后端校验并返回登录结果
- **注册流程**：
  1. 学生输入学号/姓名/密码发起注册
  2. 后端校验学号唯一性，写入 `users` 表
  3. 注册成功后可立即使用登录功能
- **角色**：学生（默认）、教师（固定1位，通过学号或后台指定）
- **个人资料**：姓名、头像、基本信息由用户自行维护

### 3.2 课程
- **单课程**：固定课程信息（名称、学期、简介）
- **公告**：教师发布课程公告，学生查看（迭代2）

### 3.3 作业/成绩
- **作业发布**：教师发布作业（题目、描述、附件、截止时间）
- **作业提交**：学生提交（文本/文件），支持查看提交状态
- **批改**：教师评分+评语，成绩查询与统计

### 3.4 出勤
- **考勤任务**：教师创建考勤（扫码/定位签到）
- **签到/签退**：学生签到、请假申请
- **出勤统计**：个人/班级统计，导出（迭代2）

### 3.5 讨论/通知（迭代2）
- **讨论帖**：学生/教师发帖、评论
- **公告推送**：课程公告通知
- **系统通知**：作业截止提醒、成绩发布等

### 3.6 统计/演示
- **学生**：作业完成度、出勤率、学习概览
- **教师**：班级出勤/作业完成率、成绩分布
- **周报与PPT**：迭代进度汇报


## 4. 账号与安全
- 登录注册接口需限制频率并对错误信息做脱敏
- 用户密码先以明文保存，后续可引入加密（BCrypt）
- 教师账号由管理员在数据库中预置，禁止页面注册教师

## 5. 文档与协作
- `APIDpocument.md` + `API.json`：接口说明 + OpenAPI，可导入 ApiFox/Swagger
- `SqldDesign.md`：核心表结构，方便初始化数据库
- 迭代周报、测试用例、PPT/演示视频用于课程汇报
