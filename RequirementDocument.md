# 项目管理与过程改进课程 - 需求简表

## 1. 项目概述
| 维度 | 说明 |
| --- | --- |
| 项目 | 项目管理与过程改进课程教学平台 |
| 服务对象 | 1位教师、1个班级的学生 |
| 范围 | 课程信息、作业与出勤、讨论互动、统计展示 |
| 技术 | Vue 3 + Vite + Element Plus / Spring Boot 3.5.6 + MyBatis Plus / MySQL 8.0 |
| 登录方式 | 对接学校教务处SSO（学号+密码） |

## 2. 迭代安排
| 迭代 | 时间 | 目标 | 关键交付 |
| --- | --- | --- | --- |
| 迭代 1 | Day 1-14 | 学生端（SSO登录→课程→作业→签到） | 前/后端可运行、SSO对接、核心SQL、Swagger/ApiFox、≥10条测试用例 |
| 迭代 2 | Day 15-28 | 教师后台 + 讨论区 + 公告 | 教师批改、考勤管理、讨论区与公告、统计导出、PPT/演示视频 |

## 3. 功能模块

### 3.1 认证/用户
- **SSO登录**：对接 `https://ids6.usst.edu.cn/authserver/login`，使用学号和密码登录
- **登录流程**：
  1. 前端跳转到学校SSO登录页
  2. 用户输入学号+密码完成认证
  3. SSO回调返回用户信息（学号、姓名等）
  4. 后端接收回调，查询/创建本地用户记录
  5. 生成JWT token返回前端
- **角色**：学生（默认）、教师（固定1位，通过学号/配置识别）
- **个人资料**：姓名、头像、基本信息（从SSO同步或手动维护）

### 3.2 课程
- **单课程**：固定课程信息（名称、学期、简介）
- **公告**：教师发布课程公告，学生查看（迭代2）

### 3.3 作业/成绩
- **作业发布**：教师发布作业（题目、描述、附件、截止时间）
- **作业提交**：学生提交（文本/文件），支持查看提交状态
- **批改**：教师评分+评语，成绩查询与统计

### 3.4 出勤
- **考勤任务**：教师创建考勤（扫码/定位签到）
- **签到/签退**：学生签到、请假申请
- **出勤统计**：个人/班级统计，导出（迭代2）

### 3.5 讨论/通知（迭代2）
- **讨论帖**：学生/教师发帖、评论
- **公告推送**：课程公告通知
- **系统通知**：作业截止提醒、成绩发布等

### 3.6 统计/演示
- **学生**：作业完成度、出勤率、学习概览
- **教师**：班级出勤/作业完成率、成绩分布
- **周报与PPT**：迭代进度汇报


## 4. SSO对接说明
- **SSO地址**：`https://ids6.usst.edu.cn/authserver/login?service={回调URL}`
- **回调处理**：后端接收SSO返回的ticket/用户信息，验证后创建/更新本地用户
- **用户同步**：首次登录自动创建用户记录，后续登录更新最后登录时间
- **教师识别**：通过配置的教师学号或角色标识区分教师/学生
