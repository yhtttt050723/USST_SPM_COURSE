# 项目计划

## 迭代 1（Day 1~14）——最小可运行版本（MVP）

### 🎯 目标
- 实现学生端核心流程，实现前后端跑通，具备基本可演示的功能。

### ⚙ 要实现的功能
- 用户登录（学生/老师）
- 课程首页（学生）
- 作业列表、作业提交
- 考勤查看
- 数据库搭建
- 后端项目结构、前端 Vue 工程搭建
- 前后端联调成功

### 👥 任务分配（按人）
- **前端 A（Vue）**
  - 登录/注册页面
  - 学生首页
  - 考勤 UI
  - Axios 全局封装
- **前端 B（Vue）**
  - 作业列表页面
  - 作业上传组件
  - 公共组件（header/nav）
- **后端 A（Spring Boot）**
  - 用户模块（登录、JWT/Session）
  - 登录权限拦截器
- **后端 B（Spring Boot）**
  - 作业 CRUD
  - 考勤查询接口
  - MyBatis Plus / JPA
- **测试 QA**
  - 登录/作业/考勤测试用例
  - 前后端联调测试
  - Bug 记录

### 📦 迭代 1 交付物
- 可运行的前端项目（支持登录 & 作业提交）
- 可运行的后端（Spring Boot + MyBatis/JPA）
- 数据库表结构 & SQL
- Postman/Swagger 接口文档 (本人用的ApiFox 不过都是差不多的软件)
- 测试用例（10+）
- 迭代 1 周报（进度 + 风险 + 问题）
